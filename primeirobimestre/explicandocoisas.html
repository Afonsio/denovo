div

A tag div não tem um significado semântico específico, o que significa que ela não indica o tipo de conteúdo que contém. Em vez disso, é usada principalmente para ajudar a organizar e estilizar o conteúdo da página usando CSS (Cascading Style Sheets).


















span

O span é outro elemento fundamental em HTML, mas ao contrário do div, ele é um elemento de linha (inline). Enquanto o div é usado para agrupar blocos maiores de conteúdo e criar divisões em uma página da web, o span é usado para aplicar estilos a pequenas partes do texto ou outros elementos dentro de uma linha de texto.


















img

A src (source) especifica o caminho para a imagem que será exibida na página. Este deve ser o caminho do arquivo de imagem no sistema de arquivos ou a URL completa da imagem na web. O atributo alt (alternativo) fornece um texto alternativo para a imagem, que é exibido se a imagem não puder ser carregada ou se o usuário estiver utilizando um leitor de tela. É importante fornecer uma descrição significativa para acessibilidade e SEO.


















aside

A tag

é um elemento em HTML usado para representar conteúdo tangencial ou relacionado ao conteúdo principal de uma página da web. Esse conteúdo geralmente é considerado secundário em relação ao conteúdo principal, mas ainda está relacionado a ele de alguma forma. O elemento
é frequentemente utilizado para criar barras laterais, blocos de conteúdo adicional, notas de rodapé ou conteúdo relacionado em um layout de página.

















audio

A tag audio em HTML5 é usada para incorporar arquivos de áudio em uma página da web, permitindo que os visitantes reproduzam e controlem a reprodução do áudio diretamente na página, sem a necessidade de um player externo.

















canvas

A tag canvas em HTML é usada para desenhar gráficos, animações e outras imagens dinâmicas diretamente no navegador da web. É uma área retangular na qual você pode desenhar usando JavaScript. A canvas é uma tag HTML5, o que significa que é uma adição relativamente nova ao HTML.

















datalist

datalist é um elemento HTML que fornece uma lista de opções para elementos input do tipo texto. Ele permite que você defina uma lista de valores predefinidos que serão sugeridos ao usuário enquanto ele digita em um campo de entrada. Quando o usuário digita algo no campo de entrada, ele pode ver uma lista suspensa de opções que correspondem ao que ele digitou.

















details

details é um elemento HTML que é usado para criar um widget de controle expansível, muitas vezes usado para revelar informações adicionais em uma página da web. Ele permite que você crie um conteúdo que pode ser oculto ou exibido ao clicar em um título ou resumo associado.
